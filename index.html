Regret

/rǝ'gret/

noun - a feeling of sadness, repentance, or disappointment over something that has happened or been done, especially a loss or missed opportunity.

I find the best code I write comes to me suddenly, like a slap in the face, moments after I've submitted something for review. Code challenges sent for evaluation, production code submitted for code review - it doesn't matter. The act of clicking those send/submit buttons fills me with the shoulda-coulda-wouldas.

This challenge was no exception, but experience has helped me develop a two-stage coping strategy. Stage one involves eating a lot of sugar and ugly-crying on my dog. Stage two is taking a crack at refactoring.

So I'm going to tackle a few of the larger items I was agonizing over.

What's Changed

New utilities module

Both the thumbnail and lightbox module had the exact same _bind method. Not very DRY.

I decided to create a separate utilities module, which could house common code. I moved the _bind, _unbind, and _preventDefaultEvent methods into this new module, because those all seemed like good candidates for things you might want to do in several places.

createNewElement Utility

Since making new DOM elements, adding classes, and sometimes adding attributes was something I was doing over and over and over in the lightbox module, I decided to create a utility for that as well.

Apply money to mouth
